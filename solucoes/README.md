# 💡 Soluções da Comunidade

Esta pasta é reservada para soluções enviadas via Pull Request.

No momento, **não há soluções publicadas** no repositório. Para contribuir:

1. Crie a pasta `solucoes/<desafio>/<linguagem>/`
2. Inclua um `README.md` com:
   - Como rodar a aplicação
   - Como rodar testes
   - Decisões de design
3. Atualize o `SOLUTIONS.md` do desafio com o link da sua solução
4. Abra um Pull Request

Se preferir compartilhar externamente, use as **Discussions**:
https://github.com/Francisco-Montalvao/backend-challenges/discussions

Para detalhes, veja [CONTRIBUTING.md](../CONTRIBUTING.md).
